**Surdoc社区、API文档及应用开发计划**

计划分为三个部分：

* 社区
* 文档
* 应用开发

计划围绕 github 进行，充分利用 github 提供的基础设施。唯一的问题，可能是其英文界面对某些用户造成轻微的困难，但对开发者来说，应该不是问题。

下面分别详述之。

# 社区

社区现行，可以在其基础上形成并完善API文档。

社区面向两类用户，普通用户和应用开发者。

github 的基础设施的使用：

*	repository
*	wiki
*	issues

# 文档

可以同时支持中文和英文两个文种。

*	FAQ
*	Tutorial
*	Reference



# 应用开发

放一到两个自己组织的应用的代码。其他开发者的代码以链接的形式提供。
